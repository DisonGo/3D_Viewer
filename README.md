# 3D_Viewer 
Implementation of software for viewing wireframe 3D objects.

## Build
### Preparation
   To start build you need to move to `./src` dir.
   ```Bash
   cd ./src
   ```
### Build
* Build program

   ```
   make all
   ```
   or
   ```
   make install
   ```
*  Build and run tests
   ```
   make tests
   ```
*  Clean project
   ```
   make clean
   ```
*  Unistall Project
   ```
   make uninstall
   ```
*  Open HTML file
   ```
   make dvi
   ```
*  Create archive with program
   ```
   make dist
   ```
